Stateful

The block, the elements and the states.

Nude uses a methodology inspired by BEM but uses state classes instead of modifiers. This way the logical hierarchy is maintained - without the long class names. 

<div class="card is-large">
    <div class="card__title is-red">
        Cards are cool
    </div>
</div>

Flexible

Responsive web design is no longer a thing

User flow should be device dependent - flexbox makes it a breeze. With the help of some media queries and the Nude grid, positioning items is never a hassle. 

<div class="cells are-gapped">
    <div class="cell is-twelve">
        <h1>Sweet!</h1>
    </div>
</div>

Lightweight

Intuitive and neutral in its presence

Web-development doesn't come without cowboy coding. Nude is built for rapid development so that all components always feel close-at-hand. 

<div class="box is-padded">
    <p class="text is-huge">
        <h1>Are you ready?</h1>
    </p>
</div>
