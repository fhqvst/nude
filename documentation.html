Introduction

Before you dive in

To get the most out of Nude there is one single rule to follow. Always let state classes describe appearance and never hierarchy. BEM takes care of the hierarchy, which means state classes like is-primary or is-secondary will never be needed. Use instead names like is-large or is-shy.

When to introduce new blocks

The answer to this question lies in two criteria: If the block will be repeated or is heavily styled. If the new block doesn't fulfill both these criteria, then introducing a new block will only lead to unwanted complexity. Nude is built with just enough blocks and state classes for these kind of situations.

Base

The base part of Nude consists of mixins, settings and a reset.
